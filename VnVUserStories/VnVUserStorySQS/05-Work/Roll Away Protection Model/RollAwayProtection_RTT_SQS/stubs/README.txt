THIS IS THE DIRECTORY WHERE TO LOCATE GLOBAL STUB FILES

Suffixes:
     .stubs : Stub Signatures
     .cnl   : Channel definition files
     .c     : C files that should be compiled into ./stubsrc/libstubglobal.a
     .h     : H files that should be used in ./stubsrc/libstubglobal.a

Note: .cnl, .c, and .h files can be used 
      to overwrite generated code in ./stubsrc/
